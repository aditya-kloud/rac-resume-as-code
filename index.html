How it works

From a prompt to a polished PDF in four steps.

1

Generate LaTeX from AI

Ask ChatGPT, Claude, or Gemini to write your resume in LaTeX. Use the supported commands reference below for best results.

2

Paste into Editor

Copy the generated LaTeX and paste it into the RaC editor. The live preview renders it instantly — no compile needed.

3

Edit & Style

Tweak content directly in the editor. Switch colour presets, change the font, and use page boundary guides to fine-tune layout.

4

Download PDF

Click Download PDF. Get a pixel-perfect A4 PDF with full-bleed colours and proper margins — ready to send.

Prompt
Write a professional LaTeX resume using \documentclass[11pt,a4paper]{article}. Use \section*{} for headings, \textbf{} for bold, \textit{} for italic, \hfill for right-aligned dates, \begin{itemize} for bullet points, and \newpage for page breaks. Include sections for Summary, Experience, Skills, and Education. My details: [paste your info here]
